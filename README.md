# 外卖点餐系统 java483

## 项目概述

外卖点餐系统是一个基于Spring Boot和Vue技术栈的全栈项目，采用前后端不分离的开发模式。本项目涵盖了管理员、商家、用户和骑手四大功能模块，实现了从菜品管理到订单配送的全流程功能。

## 技术栈

- 开发工具：IDEA、Eclipse
- 编程语言：JAVA
- 数据库：MySQL 5.7+
- 框架：SSM（Spring、Spring MVC、MyBatis）
- 构建工具：MAVEN

## 运行环境

- JDK 1.8+
- MySQL 5.7+
- Maven 3.6+

## 部署步骤

1. 准备开发环境：安装JDK、MySQL、Maven以及IDEA或Eclipse。
2. 导入SQL文件，初始化数据库。
3. 使用Maven构建项目。
4. 运行项目，进行部署。
